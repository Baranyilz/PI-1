Detailed Description

Basisklasse für sämtliche Medien.

Diese Klasse stellt eine Basisklasse für Objekte dar, die Medien beschreiben Von dieser Klasse werden alle anderen Medien, wie z.B. Buch oder DVD abgeleitet
